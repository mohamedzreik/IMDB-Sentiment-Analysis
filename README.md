# Sentiment Analysis with PyTorch RNN

Binary sentiment classification of IMDB movie reviews using a simple RNN in PyTorch.

## Dataset
- **IMDB Movie Reviews**: 25,000 reviews (12,500 positive + 12,500 negative)
- **Task**: Classify reviews as positive (1) or negative (0)

## Model
- **Embedding Layer**: 100-dimensional word vectors
- **RNN Layer**: Single-layer, 32 hidden units
- **Output**: Binary classification via fully connected layer
